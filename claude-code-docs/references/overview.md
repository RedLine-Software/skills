# Overview

Claude Code is an agentic command-line tool that helps you understand, write, and maintain codebases. It operates in an agentic loop, using tools to interact with your environment.

## Key Features

- **Agentic Workflows**: Claude Code can take a high-level goal and break it down into a plan of action, executing tools to accomplish the goal.
- **Whole-codebase Awareness**: Claude Code can read from your entire codebase, not just the files you have open.
- **Interactive and Headless Modes**: Use Claude Code in an interactive chat session, or run it programmatically in scripts and CI/CD pipelines.
- **Extensible**: Claude Code can be extended with custom skills, hooks, and MCP servers to connect to your own tools and data sources.
