# Built-in commands

This is a reference for the built-in commands available in Claude Code's interactive mode.

| Command | Description |
| :--- | :--- |
| `/help` | Show available commands |
| `/clear` | Clear conversation history |
| `/compact` | Manually trigger context compaction |
| `/cost` | View current session token usage and cost |
| `/memory`| View and edit what Claude has learned about the project. |
| `/undo`| Rewind to a previous checkpoint. |
| `/btw` | Ask a side question without adding it to the main conversation history. |
| `/model` | Switch the model for the current session. |
| `/hooks` | List and manage active hooks. |
| `/skills` | List and manage installed skills. |
| `/agents` | List and manage available sub-agents. |
| `/plugin` | Manage plugins and marketplaces. |
| `/statusline` | Configure the terminal status line. |
| `/mcp` | List and manage MCP servers. |
| `/doctor` | Check for common configuration issues. |
| `exit` | Exit Claude Code |
